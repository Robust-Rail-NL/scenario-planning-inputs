# Simple Service location

This folder contains a simple location five tracks, of which one is a servicing tracks, and the other four can be used to arrive and depart from. The [location.png](location.png) file visualizes the location as below.

![Location visuzalization](location.png)

There is a [scenario](scenario.json) with a cleaning step, that also has the associated [configuration file](scenario_config_train_cleaning.json), and solver format: [scenario_solver.json](scenario_solver.json).

There is another scenario wihtout any servicing tasks: the [scenario_no-service_solver.json](scenario_no-service_solver.json).
